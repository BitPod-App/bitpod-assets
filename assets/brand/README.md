# BitPod App Brand Starter Package

This folder contains a working starter brand system for BitPod App.

## Included
- SVG logo marks
- SVG lockups
- square avatar variants
- mini brand guidelines
- Figma master file handoff notes

## Root paths
- `assets/brand/logo/svg`
- `assets/brand/guidelines`
- `assets/brand/figma`
- `assets/brand/social/repo-previews`

## Important
This is a starter package for an in-progress product, not a final immutable brand identity.
