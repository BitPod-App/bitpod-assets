# BitPod App — Corrected Brand Starter Package

This package replaces the earlier starter bundle that drifted too far from the real logo.

## Included
- exact transparent PNG logo mark
- SVG wrappers based on the real approved logo
- square avatar variants
- horizontal and vertical lockups
- corrected guidelines and Figma notes

## Important
These SVGs are **visually exact but not true editable vector reconstructions**.
They are suitable as a practical stopgap until a proper vector redraw is created.
