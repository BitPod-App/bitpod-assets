# BitPod App — Corrected Starter Brand Guidelines

## Important
This package was rebuilt from the **actual current logo image** you approved.

Because the original deliverable did not preserve the exact vector geometry, the SVG assets in this package are **SVG wrappers containing the exact raster logo** or raster-derived variants.

That means:
- the visual identity now matches the real logo
- the files are workable for GitHub, social, docs, and Figma import
- this is **not yet a true editable vector master**

## What is reliable in this package
- exact logo mark match
- square avatar match
- horizontal and vertical lockup exports
- white / black raster-derived variants
- Figma-importable SVG wrappers
- transparent PNG master mark

## What still needs a future upgrade
A true vector redraw of the approved logo, created manually in Figma or Illustrator.

## Recommended current usage
Use:
- `bitpod-logo-mark-exact.png` or `.svg` for most brand use
- `bitpod-logo-avatar-square-color.png` for GitHub / avatars
- horizontal lockup for headers / banners
- vertical lockup for centered layouts

## Fonts
Recommended:
- Inter
Fallback:
- Arial / sans-serif

## Colors
- Background navy: `#0C122C`
- Main text / off-white: `#F4EFE8`
- Muted text: `#B9BFCE`

The approved logo itself carries its own internal gradient and should be treated as the source of truth.

## Corrective note
This package replaces the earlier unusable starter set that drifted too far from the real approved logo.
