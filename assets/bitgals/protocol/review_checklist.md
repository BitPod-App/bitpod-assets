# BitGal Review Checklist

Use this fast checklist before an asset enters the official library.

## 1. Same base woman?
- Yes / No

## 2. Right persona read?
- Taylor / Orange / Kati / Shiva / Vera / Wrong

## 3. Required anchors present?
- Check against `bitgal_canon_minimal_v1.yaml`

## 4. Forbidden traits present?
- If yes, usually fail

## 5. Scene fit correct?
- HQ / stage / studio / night / office-tech / other

## 6. Quick score
- Base identity: /100
- Persona: /100
- Confusion: /100
- Total: /100

## Decision
- Approved
- Conditional
- Rejected

## Hard-fail triggers
- not the same base woman
- wrong persona read
- missing required anchor
- generic AI portrait slop
