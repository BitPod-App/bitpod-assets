# Published Assets

Operational asset lane.

Use this folder for assets that are:

- already live
- already wrapped by Cloudflare
- ready to be used at any time
- intended for direct product/app/site use

This folder is intentionally separate from `bitgals/`.

`bitgals/` is the identity-canon and review machine.
`published/` is the live/hostable/use-now machine.

Published persona assets may include still images, animated GIF previews, and
other media formats when they are intended for direct product use. Promotion to
Cloudflare delivery must be explicit in the relevant manifest and registry.

## Current top-level buckets

- `branding/`
- `personas/`
- `bitgals/`

Keep this structure simple and obvious.

## Rule

An asset can be live here without having passed through the full `bitgals/`
identity-canon workflow.

This folder tracks what is operationally usable.
