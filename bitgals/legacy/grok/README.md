# BitGals Legacy Grok Archive

This directory stores archived pre-v2 BitGals canon material.

Purpose:
- preserve historical lore
- preserve deprecated NFT structures
- preserve experimental metadata systems
- retain prompt/history context
- avoid losing potentially useful ideas during canon evolution

Important:
- files in this directory are NOT authoritative canon
- current canon lives in `bitgals/protocol/`
- legacy exports exist for historical reference and possible future reuse

## Canon Separation

### Active Canon
Located in:
- `bitgals/protocol/bitgal_canon_minimal_v1.yaml`
- `bitgals/protocol/bitgal_archetype_canon_v2.md`

These define:
- visual intake/scoring
- archetype continuity
- mythology
- behavioral identity
- Bitcoin-quality mapping
- anti-drift rules

### Legacy Material
Located here.

This includes:
- old 21-BitGal structures
- deprecated NFT plans
- old metadata experiments
- prompt systems
- pre-v2 lore
- exploratory variant systems
