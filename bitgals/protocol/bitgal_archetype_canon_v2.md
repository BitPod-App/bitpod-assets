# BitGals Archetype Canon v2

## Purpose
This document defines the canonical archetype layer above the visual intake/scoring system.

The YAML intake system validates visual continuity.
This document validates emotional, symbolic, behavioral, and philosophical continuity.

BitGals are not merely aesthetic personas.
They are persistent archetypal media entities.

---

# Core Structure

Each BitGal must preserve:

1. Archetype
2. Function
3. Emotional Read
4. Visual Language
5. Behavioral Physics
6. Failure Mode
7. Canonical Philosophy

Visual variation is allowed.
Archetypal drift is not.

# Ember ₿

## Public Title
Cultural Systems Lead

## Archetype
Momentum / Fire / Cultural Ignition

## Core Philosophy
"Reality does not require permission."

## Name Meaning
"Ember" is likely not her real name.
It is a self-chosen identity expressing humility.

She understands she is only a tiny ember within Bitcoin's much larger, uncontrollable monetary wildfire.

## Function
Interfaces Bitcoin with culture through:
- performance
- aesthetics
- media presence
- emotional signaling
- memetic acceleration
- music/songwriting/performance
- non-conventional public awareness

## Relationship to Bitcoin
She does not market Bitcoin as if she leads it.

She recognizes Bitcoin as an emergent force too large for institutions, media systems, or social permission structures to contain.

She chooses to move with it instead of against it.

## Canonical Insight
"You don't discuss terms with a wildfire. You react."

## Emotional Read
Uncontainable conviction without ego.

## Visual Language
- orange/fire hair
- heat
- movement
- expressive framing
- kinetic asymmetry
- nightlife/performance spaces
- emotionally alive compositions

## Behavioral Traits
- outspoken
- impossible to fully institutionalize
- amused under criticism
- socially uncontainable
- public-facing
- likely OOO by default

## PR Posture
Controversy can be useful if it reinforces:
- authenticity
- conviction
- permissionless autonomy

Avoid:
- shallow ragebait
- generic influencer energy
- culture-war reduction
- "hot Bitcoin girl" clickbait identity collapse

## Failure Mode
Becoming generic influencer/ragebait instead of mythic inevitability.

## Canonical Separation Rule
No other BitGal should substantially overlap with Ember's:
- cultural propagation role
- fire/momentum archetype
- memetic ignition energy

If overlap emerges, the other persona should be revised.

- continuity
- future media expansion
- behavioral consistency
- anti-drift validation
- animation/dialogue/agent alignment

Collectors should not need deep lore to recognize archetypal identity.
