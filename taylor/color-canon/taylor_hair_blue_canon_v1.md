# Taylor Hair Blue Canon v1

## Purpose

This document defines the canonical target hair color for Taylor's electric-blue presentation.

This is a **visual identity rule**, not a loose style suggestion.

## Canonical target

Taylor's hair should read as:

- **bright electric cyan-blue**
- **cool-toned**
- **high saturation**
- **high brightness**
- **clean synthetic neon blue**
- **visibly more blue than teal**

## Plain-language anchor

The correct color impression is:

> vivid electric blue with a cyan edge, like luminous plasma-blue or neon sky-blue, without drifting into turquoise, cobalt, navy, pastel, or purple-blue.

## What must remain true

The hair should:

- read immediately as **electric blue** in normal lighting
- remain vivid even in shadow
- preserve realistic strand variation
- preserve realistic shadowing and gloss
- avoid looking painted, flat, or cartoonishly solid

## Drift that must be rejected

Reject outputs that drift toward:

- **teal**
- **turquoise**
- **greenish aqua**
- **dark cobalt**
- **navy**
- **washed-out pastel blue**
- **lavender-blue**
- **purple-blue**

## Identity rule

Changing the hair color must **not** change:

- face
- identity
- expression
- pose
- framing
- skin tone
- hair shape
- strand structure
- realism
- lighting logic

## Operational interpretation

This canon should be treated as the default Taylor blue unless CJ explicitly requests another hair color direction.

## Naming suggestion for edited assets

Use names like:

- `taylor_<variant>_electric-blue_v1.png`
- `taylor_<variant>_electric-blue_v2.png`

Where:
- `<variant>` identifies the source or pose
- version increments only when a materially improved candidate is produced
