# Repo Previews

Social preview cards used in org profile/readme surfaces.

## Naming convention

- `<repo-name>-preview.png`

Current:

- `bitpod-tools-preview.png`
- `sector-feeds-preview.png`

## Format

- Recommended size: `1280x640`
- Keep text high contrast and legible at small scale.
- Prefer dark background + disciplined orange accent for BitPod consistency.
